2.1 – Función energética base: ρ_IE(t)

La densidad energética del Modelo IE se define como una función suave y decreciente:

ρ_IE(t) = ρ_∞ + (ρ₀ − ρ_∞) / [1 + (t / t_s)^α]

Esta fórmula describe una evolución energética continua sin inflatón, donde:

  • ρ₀: densidad inicial
  • ρ_∞: densidad residual estabilizadora
  • t_s: escala temporal
  • α: exponente de caída
